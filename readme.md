# a wall of stellar transactions memo

Some web-components:
- send a transaction to an account
- list all transactions of an account

Some scripts:
- `node ./make_account.js`
- `node ./make_payment.js`

Shell Commands:
- $ `curl "https://friendbot.stellar.org/?addr=ACCOUNT_ID"`
